# Images Directory

Tento adresár obsahuje obrázky pre web.

## Požadované obrázky:

1. **laptop-tech.jpg** - Obrázok laptopu s holografickými dátovými vizualizáciami (pre mobile)
   - Rozmery: cca 800x600px alebo väčšie
   - Formát: JPG/PNG
   - Použitie: Zobrazuje sa na mobile v Hero sekcii

2. **planet-tech.jpg** - Obrázok planéty s modrými prstencami (pre desktop)
   - Rozmery: cca 1200x1200px alebo väčšie
   - Formát: JPG/PNG
   - Použitie: Zobrazuje sa na desktop v Hero sekcii

## Ako pridať obrázky:

1. Stiahnite obrázky z poskytnutých zdrojov
2. Premenujte ich na `laptop-tech.jpg` a `planet-tech.jpg`
3. Uložte ich do tohto adresára (`public/images/`)
4. Obrázky budú automaticky dostupné na `/images/laptop-tech.jpg` a `/images/planet-tech.jpg`
